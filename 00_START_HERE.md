## 🎓 Key Concepts

### How It Works
1. **Create content in Sanity Studio** (visual, no code)
2. **Frontend fetches via GROQ queries** (JavaScript)
3. **Content renders to website** (HTML/CSS)
4. **Users see live website** (GitHub Pages)

### Why This Architecture
- ✅ No backend servers needed
- ✅ Content changes are instant
- ✅ Easy to manage with Sanity CMS
- ✅ Free hosting on GitHub Pages
- ✅ Fully responsive design
- ✅ Professional look

### Technology Stack
```
Frontend: HTML5 + CSS3 + JavaScript
Sanity: Headless CMS (cloud-hosted)
Hosting: GitHub Pages (free, fast)
```
